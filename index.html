To access our server's custom mods, please follow these instructions:

  1. Download the mod package using the button provided below.
  2. Ensure you have the latest edition of Forge installed for Minecraft version 1.20.1.
  3. Open Minecraft and navigate to the "Mods" section on the main menu.
  4. Locate the mods folder within Minecraft and copy its file path.
  5. Extract the contents of the downloaded zip file into the mods folder, ensuring to preserve the directory structure.
  6. Launch Minecraft, and you will now have access to the installed mods for enhanced gameplay experience.
